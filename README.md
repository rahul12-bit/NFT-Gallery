 🖼️ NFT Gallery (Soroban Smart Contract)

## 📌 Project Description
NFT Gallery is a decentralized smart contract built using Soroban on the Stellar network. It allows users to mint, transfer, and view NFTs in a simple on-chain gallery system.

This project demonstrates how to build and manage NFTs using Stellar’s smart contract platform.

---

## 🚀 What it does
- Allows users to mint NFTs with unique IDs and metadata
- Stores NFT ownership securely on-chain
- Enables transfer of NFTs between users
- Provides a way to view individual NFTs and list all NFTs

---

## ✨ Features
- 🔐 Secure ownership using wallet authentication
- 🪙 NFT minting functionality
- 🔄 Transfer NFTs between users
- 📄 Metadata support (basic symbol-based metadata)
- 📋 View and list all NFTs in the gallery
